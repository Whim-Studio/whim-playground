# Balance reports

`--sim` writes `balance.csv` and `balance.json` here.

> The `balance_sample.*` files are **ILLUSTRATIVE placeholders** showing the
> report *shape* only. They were NOT produced by an executed run (this module
> was authored without a JDK available). Regenerate real numbers with:
>
> ```bash
> java -jar target/babylon5-deckbuilder.jar --sim --games 1000 --players 4 --seed 42 --ai hard
> ```
